# 主张
- R001：若上下文存在 `SpecificChannelId`，路由直接使用该渠道；渠道不存在或已禁用则拒绝请求。

# 证据
- 文件：`internal/transport/http/middleware/auth.go`
  - 函数/符号：`TokenAuth`
  - 行号范围：L335-L348
  - 片段概述：管理员 token 后缀或 `channelid` 参数写入 `ctxkey.SpecificChannelId`。
- 文件：`internal/transport/http/middleware/distributor.go`
  - 函数/符号：`Distribute`
  - 行号范围：L28-L43
  - 片段概述：若 `SpecificChannelId` 存在则 `GetChannelById`，并检查 `ChannelStatusEnabled`；否则拒绝。

# 结论
- 指定渠道优先于随机选渠，且禁用渠道会被直接拦截。
